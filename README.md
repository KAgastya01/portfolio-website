# Agastya Portfolio Website

This is a dependency-free static portfolio website built from the attached PDF resume.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive layout and visual design
- `script.js` - mobile navigation, active nav state, and copy-email behavior
- `assets/hero-visual.png` - generated bitmap hero visual

## Run In VS Code

1. Open VS Code.
2. Go to `File > Open Folder`.
3. Select this folder:
   `C:\Users\91934\Documents\Codex\2026-06-05\files-mentioned-by-the-user-creative\outputs\agastya-portfolio`
4. Open `index.html`.
5. Right-click inside the file and choose `Open with Live Server`.

If you do not have Live Server:

1. Install the VS Code extension named `Live Server` by Ritwick Dey.
2. Reload VS Code if prompted.
3. Right-click `index.html` and choose `Open with Live Server`.

You can also double-click `index.html` to open it directly in a browser because the site does not require Node, npm, or a build step.

## Customize

- Update contact details in `index.html` inside the `#contact` section.
- Add real project links by replacing the project card copy in the `#work` section.
- Replace `assets/hero-visual.png` with your own image if you want a personal photo or project screenshot.
